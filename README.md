> VM(Virtual Machine, 가상머신)은 각 VM 단위로 서로 다른 운영체제(Guest OS)를 설치할 수 있는 장점이 있지만, 
같은 운영환경에서 실행되는 여러 서비스의 경우 이 장점은 오히려 손님 운영체제(Guest OS)의 중복으로 단점이 된다.    
또한, 관리 방법도 Docker에 비해 많이 어렵고 복잡하다.
